# Lisp_Interpreter

Lisp interpreter, written in Python.

To Run: python3 LispInterpreter.py

## Supports the following functionalities of Lisp:
{IF}, {WHILE}, {SET}, {BEGIN}, Operators: {+}, {-}, {*}, {/}, {=}, {<}, {>}, {CONS}, {CAR}, {CDR}, {NUMBER?}, {SYMBOL?}, {LIST?}, {NULL?}, {PRINT}, {T}, {Define}, {Call}
